# FaceFiltering

Python 3.10 or newer should work.

From this folder:

```
python setup_and_run_ui.py
```

That script makes a local `.venv`, runs pip against `requirements.txt`, then starts `app.py` (Gradio). Ctrl+C stops it.

If something is off with the venv, run the script again; it can recreate a broken `.venv`.
